Specification of requirements

Mandatory requirements

  • Read hand histories from file
  • Some kind of database
  • An easy-to-use user interface
  • Extract some basic stats for players based on their actions on different streets:
    - Hands played
    - Money won
    - Voluntarily put money in the pot %
    - Preflop raise %
    - Preflop 3bet %
    - Agression factor (Raise% + Bet%) / Call %
    - Continuation bet flop %
    - Went to showdown %
    - Won at showdown %
  • Filtering by date, sessions, private cards, position at the table and money won
  • Reviewing single hands
  • Plot money won over hands played (or save it in some easy-to-plot format)

Additional requirements

  • Graphical user interface
  • Calculate pot odds to improve hand review
  • More stats and features, e.g. tip a dealer, burn cards before draw, etc.
  • Use some real database system
